# 🚀 快速启动指南

## 方案1：使用启动器（推荐）

### Windows用户：

双击运行 `launcher.py`，或使用命令：

```bash
python launcher.py
```

启动器会自动：
1. ✅ 检查项目完整性
2. ✅ 检查依赖安装
3. ✅ 测试API连接
4. ✅ 启动主程序

---

## 方案2：手动步骤

### 第1步：检查项目

```bash
python setup_check.py
```

如果显示缺少依赖，运行：

```bash
python check_and_install.py
```

或手动安装：

```bash
pip install -r requirements.txt
```

### 第2步：测试API连接

```bash
python test_connection.py
```

应该显示"✅ 连接成功！"

### 第3步：启动系统

```bash
python main.py
```

---

## Windows用户（如果python命令不工作）

尝试以下命令：

```bash
python3 main.py
```

或

```bash
py main.py
```

## 常见问题

### Q: 提示找不到python命令

**A:** 尝试使用 `python3` 或 `py` 命令

### Q: 依赖包未安装

**A:** 运行 `pip install -r requirements.txt`

### Q: API连接失败

**A:**
1. 检查 `.env` 文件是否存在
2. 确认 DEEPSEEK_API_KEY 已正确配置
3. 检查网络连接

### Q: Stable Diffusion未安装

**A:** 图片生成是可选的，未安装时系统只会生成文案，不影响使用
