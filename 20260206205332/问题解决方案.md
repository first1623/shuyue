# ❓ 问题：Python命令无法识别

## 📋 问题原因

您的系统中没有安装Python，或者Python没有添加到系统PATH环境变量。

---

## ✅ 解决方案（按推荐顺序）

### 🥇 方案1：通过Microsoft Store安装（最简单）

**步骤：**

1. 按 `Win + S` 打开搜索
2. 搜索 "Microsoft Store"
3. 打开 Microsoft Store
4. 搜索 "Python"
5. 点击 "Python 3.11"（或最新版本）
6. 点击 "获取" 或 "安装"
7. 等待安装完成

**验证：**
关闭所有命令窗口，重新打开PowerShell，输入：
```powershell
python --version
```

如果显示版本号，安装成功！

**启动系统：**
```powershell
cd C:\Users\zhaoy\CodeBuddy\20260206205332
python launcher.py
```

---

### 🥈 方案2：从官网安装（最稳定）

**步骤：**

1. 访问 https://www.python.org/downloads/
2. 点击 "Download Python 3.11.x"（或最新版本）
3. 下载安装程序
4. **重要：** 运行安装程序时，**第一屏必须勾选** "Add Python to PATH"
5. 点击 "Install Now"
6. 等待安装完成
7. 重启电脑

**验证：**
打开PowerShell，输入：
```powershell
python --version
```

**启动系统：**
```powershell
cd C:\Users\zhaoy\CodeBuddy\20260206205332

```

## ⚠️ 常见错误及解决

### 错误1：'python' 不是内部或外部命令

**原因：** Python未安装或未添加到PATH

**解决：**
- 按照方案1或2安装Python
- 确保安装时勾选"Add Python to PATH"
- 重启电脑

### 错误2：pip不是内部或外部命令

**原因：** Python的Scripts目录未添加到PATH

**解决：**
```powershell
python -m pip install --upgrade pip
```

### 错误3：找不到模块

**解决：**
```powershell
pip install -r requirements.txt
```
