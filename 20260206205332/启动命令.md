## 🔍 如果遇到问题

### 问题1：提示"找不到python命令"

**解决方案：**

#### 方案A：使用py启动器（推荐）
```powershell
py launcher.py
```

#### 方案B：安装Python
1. 访问 https://www.python.org/downloads/
2. 下载并安装Python 3.8+
3. 安装时勾选"Add Python to PATH"

#### 方案C：使用已安装的Python
查找Python安装路径：
```powershell
where python
```

然后使用完整路径：
```powershell
"C:\你的Python路径\python.exe" launcher.py
```

### 问题2：依赖包未安装

**解决方案：**

```powershell
pip install openai python-dotenv requests Pillow
```

或使用requirements.txt：
```powershell
pip install -r requirements.txt
```

### 问题3：权限错误

**解决方案：**

以管理员身份运行PowerShell：
1. 搜索"PowerShell"
2. 右键 > 以管理员身份运行
